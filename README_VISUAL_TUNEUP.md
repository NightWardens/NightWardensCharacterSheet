# Night Wardens Website Visual Tune-Up Package

This package combines the Digital Field Office, Auto-GM page, storefront, digital library, admin placeholder, Firebase files, and the uploaded visual/UI art as local compressed WebP assets.

## Upload to GitHub Pages
Upload everything in this folder to the root of the GitHub Pages repository. Keep the folder structure intact.

## Key files
- index.html — Digital Field Office / character app
- auto-gm.html — Auto-GM field guide console
- storefront.html — product storefront
- digital-library.html — claim/access page
- admin.html — local/Firebase-ready product admin
- visual-assets.html — preview gallery for included images
- assets/visual-theme.css — visual tune-up override layer
- assets/images/ — compressed uploaded images plus generated product covers

## What changed
- Dark dossier visual system applied across the site.
- Uploaded art compressed into web-ready local assets.
- Product cover placeholders generated from the included Night Wardens art.
- Store products now point to local cover files.
- Navigation between Field Office, Storefront, Library, Auto-GM, and Admin was added.

Review firebase-config.js and your Firebase rules before public upload.
