# Night Wardens Commercial Blueprint Page

This package adds a production-ready standalone page to your existing GitHub Pages site.

## Files

Upload these to the root of your repo:

- `commercial-blueprint.html`
- `assets/nw-commercial-blueprint.css`
- `assets/nw-commercial-blueprint.js`

Optional snippet:

- `snippets/add-nav-link.html`

## URL

After upload, open:

`https://YOUR-GITHUB-ACCOUNT.github.io/NightWardensCharacterSheet/commercial-blueprint.html`

Because all links are relative, this will still work if you later change your GitHub account or organization name to something like NightWardens.

## Suggested navigation link

Add this to your existing navigation:

```html
<a href="commercial-blueprint.html">Commercial Blueprint</a>
```

## Notes

- This page is static HTML/CSS/JS and does not need a build step.
- It does not overwrite your existing `index.html`.
- It uses the same grounded dossier / occult field office visual direction.
- Buttons are organized around the main blueprint sections instead of fake checkout links.
